# 同源策略
同源策略限制从**一个源**加载的文档或脚本如何与来自**另一个源的资源**进行交互。这是一个用于隔离潜在恶意文件的关键的安全机制。

一个源的定义
如果**协议**，**端口**（如果指定了一个）和**域名**对于两个页面是相同的，则两个页面具有相同的源。
